Sequence element class

Test fixture initialization::

    >>> from dragonfly import *

Sequence
----------------------------------------------------------------------------

Basic usage::

    >>> seq = Sequence([Literal("hello"), Literal("world")])
    >>> test_seq = ElementTester(seq)
    >>> test_seq.recognize("hello world")

    >>> test_seq.recognize("hello universe")
    RecognitionFailure
